# 入门步骤

# 安装 （win 10）

  • 直接下载 SDK zip

  • 直接克隆 git clone -b stable https://github.com/flutter/flutter.git （如果是下载的release，后续操作也会报错，提示用克隆方式）

将 bin 目录放到环境变量中

  • 在命令行或power shell（不要在git bash中）运行 flutter doctor 需要等待较长时间，然后会提示有哪些问题

  • 安装Android Studio
